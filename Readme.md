# Drink 2 Go
#### _online store of coffee drinks_
#
- HTML / SCSS / JS / Gulp
- BEM methodology
- Mobile first
- Loading otimization
- Retina adaptive
- Crossbrowser
- Semantic & responsive layout
- Leaflet.js
- Swiper.js

## Installation & start

```sh
cd drink2go
npm i
npm start
```
